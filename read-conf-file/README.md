# read-conf-file

Clojure solution for Rosetta Code 'Read a configuration file' task.

http://www.rosettacode.org/wiki/Read_a_configuration_file

## Usage

    $ java -jar read-conf-file-0.1.0-standalone.jar resources/config.txt
